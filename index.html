What Are Wireframes?

Wireframes are a drawing/sketch of the basic outline of the webpage you are going to create. It is used to give each member the idea of what the end result should look like, so they are all on the same page. As long as each member follows the wireframe they all agreed on using, there will be less confusion when making the webpage.

Git Workflow

When in a group with a bunch of other programmers, each member has to have the same code to prevent any issues. A member can push up their code from their own branch using a pull request, where the manager will then look over it and merge it with the master branch if it is all good. All the other programmers need to then pull that new code from the master brance to their own branch/code. If they try to merge code that dosent have everything that the master branch has, issues will occure.

Ways to Create Responsive Web Design

  • Using Percentages: When givng an elements such as a div a height and width, using percentages instead of px will make that div size change depending on the screen size, making it more responsive.
  • FlexBox: A techniqe that is used to arrange divs into different orders and can space them out in many different ways.
  • Using Meta: Meta is a useful way to make elements positioning correspond with other elements positions.
  • Using the position tag in CSS: Depending on what you set the position tag as in CSS, it can make elements more responsive. Using the relative tag is very useful because it makes that element's positions correspond with other elements positionings.
